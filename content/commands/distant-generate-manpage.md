# NAME

distant-generate-manpage

# SYNOPSIS

**distant-generate-manpage** \[**\--dir**\] \[**-h**\|**\--help**\]

# DESCRIPTION

# OPTIONS

**\--dir**=*DIR*

:   If specified, will output to the directory at the given path instead
    of stdout

**-h**, **\--help**

:   Print help information
