# NAME

distant-client-action-file-read-text - Reads a file from the specified
path on the remote machine and treats the contents as text

# SYNOPSIS

**distant-client-action-file-read-text** \[**-h**\|**\--help**\]
\<*PATH*\>

# DESCRIPTION

Reads a file from the specified path on the remote machine and treats
the contents as text

# OPTIONS

**-h**, **\--help**

:   Print help information

\<*PATH*\>

:   The path to the file on the remote machine
